# Icons

Place PNG icons here for the Chrome extension:

| File         | Size   | Use                    |
|--------------|--------|------------------------|
| `icon-16.png`  | 16×16  | Toolbar / favicon      |
| `icon-32.png`  | 32×32  | Windows toolbar        |
| `icon-48.png`  | 48×48  | Extensions management  |
| `icon-128.png` | 128×128| Chrome Web Store       |

Without these files, Chrome will show a default puzzle piece. Generate from a single 128×128 source (e.g. with ImageMagick or an online resizer).
